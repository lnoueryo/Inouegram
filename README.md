## アプリケーションの概要
インスタグラムのような写真共有ウェブアプリ
## アプリケーションの機能一覧
・認証機能
・ソーシャルログイン機能
・メール機能
・写真投稿機能
・写真加工機能
・フォロワーの写真閲覧機能
・いいね、コメント機能
・ユーザー検索機能
・プロフィール編集機能
・写真投稿削除機能
・管理者側のページネーション機能
## アプリケーション内で使用した技術
HTML/CSS/JavaScript(Vue.js)/Vuetify/TypeScript/PHP(Laravel)/Apache/MariaDB/AWS(EC2/VPC/Route53)

## 工夫したところ
・AJAXを使ったREST APIの構築
・Oauthの認証
